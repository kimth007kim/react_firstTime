3-1Elements개념과 렌더링 과정

Elements는 무엇일까? 요소 성분이라는 뜻 영어로는
리액트에서는 "React앱을 구성하는 가장 작은 블록이다."
Elements는 화면에서 보이는것들을 기술한다.

Elements의 특징
1. 불변성을 가진다. Element생성후에는 children이나 attributes를 바꿀수 없다.

2. Elements를 DOM에 렌더링 하기
Root DoM node
<div id="root"></div>		//우리는 여기다가 엘리먼트들을 렌더링할것이다.
				//이 div안에 있는 모든것들이 React DOM에 의해서 관리된다.	
				//오직 React로만 만들어진 앱들은 단하나의 ReactDOM을 가지게 되고
				//반면에 기존에 있던 앱에 리액트를 연동하게 되면, 여러개의 분리된 수많은 rootDOM을 가질수있다.

const element = <h1>Hello,world</h1>		//element 생성완료
ReactDOM.render(element,document.getElementById('root'));	//엘리먼트를 root에 렌더링한다.

	2.2 바뀐부분만 계산
	2.3 렌더링
